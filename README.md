# NumPy Explorer
This project demonstrates NumPy fundamentals: array creation, indexing & slicing, axis-wise operations,
reshape & broadcasting, saving/loading arrays, CSV I/O and performance comparison with Python lists.

## How to run
1. Install dependencies:
   pip install numpy
   pip install matplotlib  # optional for plotting

2. Run:
   python numpy_explorer.py
   python numpy_explorer.py --plot

## Files produced
- saved_arrays/explorer_data.npz : saved NumPy arrays
- sample_dataset.csv : synthetic dataset
- plot_sqrt_example.png : optional plot
